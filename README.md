## 📊 学习路径

### 🌱 入门路径（2-3 个月）

```
Month 1: AI 基础
├─ AI 基础概念
├─ 机器学习入门
└─ 深度学习基础

Month 2: RAG 系统
├─ RAG 概述
├─ Week 1-3: 基础设施和搜索
└─ Week 4-5: 混合搜索和完整 RAG

Month 3: 高级应用
├─ Week 6-7: 生产监控和 Agentic RAG
├─ LLM 基础和 Prompt Engineering
└─ Agent 开发
```

### 🚀 进阶路径（1-2 个月）

```
Week 1-2: RAG 系统深度实践
├─ Week 1-3: 跳过基础，直接从 Week 4 开始
├─ Week 4-7: 重点学习混合搜索、监控和 Agentic RAG
└─ 实现自己的 RAG 系统

Week 3-4: LLM 应用开发
├─ Prompt Engineering 实战
├─ Agent 开发实践
└─ 构建自己的 AI 应用
```
